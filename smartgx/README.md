# SmartGX Prototype

Step 1 scaffold for a React Native Expo app that prototypes SmartGX as a GXBank feature layer.

Included in this step:
- Expo Router navigation skeleton
- Theme tokens
- Global TypeScript types
- Mock datasets
- Mock auth store with demo users
- Supabase-ready env template and client placeholder

Run:
- `npm install`
- `npx expo start`
